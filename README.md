# File Merging and Compressing Website

## Project Overview
This project provides a web-based application for merging and compressing files. The aim is to simplify the process of handling multiple files by allowing users to combine them into one file and reduce their size for easier sharing and storage.

## Features
- **File Merging**: Users can upload multiple files which will be merged into a single file format of their choice.
- **File Compression**: After merging, users can compress the resultant file to reduce its size using various compression algorithms.
- **User-Friendly Interface**: An intuitive interface ensures that users can navigate the application easily.
- **Security**: Uploaded files are handled securely and deleted after processing to maintain user privacy.

## Installation
To set up this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Kamaze123/File-Merging-and-Compressing-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd File-Merging-and-Compressing-website
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm run start
   ```

## Usage
1. Open your web browser and go to `http://localhost:3000`.
2. Use the interface to upload files for merging.
3. Select the desired output format and compression settings.
4. Download the processed file once the operation is complete.
